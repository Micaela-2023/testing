# Cypress exercise

Techical test of using [Cypress](https://www.cypress.io/) end-to-end (E2E) test runner to perform a [Google](https://www.google.com/) search. 

## Prerequisites

- [Node.js](https://nodejs.org/)

## Install

Clone repository:

```sh
git clone https://github.com/
```

Install dependencies:

```sh
npm install
```

## Run

Open Cypress:

```sh
npm run cypress
```
